# RestaurantReview (React-Native)

- Make review on different restaurant by giving your personal rates

## Getting started

### General prerequisites

-   NodeJS
-   npm

### Library Used

- React native for both (IOS and Android)

### Clone the project

### Install dependencies

`npm install`

### Run the project

`npm start`
